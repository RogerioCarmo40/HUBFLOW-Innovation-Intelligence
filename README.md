# HubFlow Innovation Intelligence

O HubFlow é uma plataforma de gestão de inovação projetada para transformar o "ruído de mercado" em insights acionáveis, utilizando agentes de IA para triagem e análise de viabilidade de projetos.

## 🚀 Funcionalidades Principais
*   **Gestão de Funil de Inovação:** Acompanhamento de ideias desde a fase de *draft* até a implementação.
*   **Agentes de IA Especializados:** Ferramentas integradas para análise de mercado, riscos, viabilidade técnica e tendências.
*   **Dashboard de Analytics:** Visualização em tempo real de KPIs de inovação, incluindo taxa de aprovação e tempo médio de revisão.
*   **Ambiente Seguro:** Autenticação unificada para garantir a proteção de dados sensíveis de projetos.

## 🛠 Tecnologias Utilizadas
*   **Frontend:** React, Tailwind CSS, ShadcnUI.
*   **Backend/API:** FastAPI, Python.
*   **Banco de Dados:** Supabase (PostgreSQL).
*   **Inteligência Artificial:** Agentes autônomos integrados via API.
